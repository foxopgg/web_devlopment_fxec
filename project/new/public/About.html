About This Project

This College Database Management System is designed for institutions to digitally manage student and staff records. It enables easy entry, viewing, and updating of data, improving productivity and data accessibility.

Tech Stack: HTML, CSS, JavaScript, Node.js, MySQL

Key Features

  • Add and View Students with Subject-wise Marks
  • Add and View Staff with Department Information
  • SQL Integration for persistent data storage
  • Responsive and Animated User Interface
  • Modular codebase with API structure
  • Future-ready for extensions like login/auth, analytics, etc.
